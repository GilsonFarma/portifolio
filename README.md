# Portfólio - Gilson Pereira

Este é meu primeiro projeto de portfólio desenvolvido durante meus estudos em programação.

O objetivo do projeto é apresentar minhas habilidades iniciais em desenvolvimento web e servir como base para futuros projetos.

## Tecnologias utilizadas

- HTML
- CSS
- Flexbox
- Git
- GitHub

## Estrutura do site

O site contém:

- Apresentação pessoal
- Seção "Sobre mim"
- Lista de habilidades
- Projetos
- Contato

## Objetivo

Praticar conceitos fundamentais de desenvolvimento front-end como:

- Estrutura HTML semântica
- Estilização com CSS
- Organização de layout
- Controle de versão com Git
